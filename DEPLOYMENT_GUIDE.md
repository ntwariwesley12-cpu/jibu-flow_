## 🚀 QUICK START - DEPLOY YOUR WEBSITE ONLINE

Your website is ready to be deployed! Follow these simple steps:

---

## **OPTION 1: Deploy on Heroku (FREE & EASIEST)**

### Step 1: Create Heroku Account
- Visit: https://www.heroku.com
- Sign up for free account
- Verify your email

### Step 2: Install Heroku CLI
- Download: https://devcenter.heroku.com/articles/heroku-cli
- Install it
- Open PowerShell and verify: `heroku --version`

### Step 3: Deploy Your App
```powershell
cd "c:\Users\EliteBook\Desktop\jibu flow"
heroku login
heroku create jibu-flow-water
git init
git add .
git commit -m "Initial commit - Jibu Flow Water"
git push heroku master
```

### Step 4: Your Website Goes Live!
- Your URL: `https://jibu-flow-water.herokuapp.com`
- Or custom domain: Add domain in Heroku dashboard

## **REQUIRED ENVIRONMENT VARIABLES**

When deploying, set these variables in your hosting platform:

```
EMAIL_USER=your_gmail@gmail.com
EMAIL_PASS=your_app_password (not regular password)
TWILIO_ACCOUNT_SID=your_twilio_sid
TWILIO_AUTH_TOKEN=your_twilio_token
OWNER_PHONE_NUMBER=+250XXXXXXXXX
OWNER_EMAIL=your_email@gmail.com
```

### How to get these:
1. **Gmail App Password**: https://support.google.com/accounts/answer/185833
2. **Twilio Credentials**: https://www.twilio.com/console
